Module integrates Datacash payment gateway with Drupal Commerce module.

Installation

Download the Datacash PHP library from site https://testserver.datacash.com/software/download.cgi to module's folder.
Rename the library name to DataCash-PHP-XMLAPI. So the structure of the commerce_datacash module folder will be:

.                                  commerce_datacash.info
..                                 commerce_datacash.module
commerce_datacash.error_codes.inc  DataCash-PHP-XMLAPI
commerce_datacash.helpers.inc      README.txt
